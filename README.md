# LogSight 🔍

A production-style **System Monitoring & Alerting Platform** — built with Node.js, PostgreSQL, React, BullMQ, and the  API.



## What is LogSight?

LogSight lets you:
- **Ingest logs** from any application via a simple HTTP API
- **Monitor** error rates, response times, and service health in real time
- **Alert** when thresholds are breached (with cooldown logic to prevent spam)
- **Analyze** trends with an AI-powered insights engine (Claude API)
- **Visualize** everything on a React dashboard
